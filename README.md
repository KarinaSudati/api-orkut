



# 🌐 API Rede Social - Backend Fullstack

Esta API foi desenvolvida com **Node.js**, **Express** e **PostgreSQL**, com o objetivo de demonstrar na prática a construção de um backend robusto e escalável.

O projeto faz parte do meu aprendizado no curso Fullstack e foca em segurança, persistência de dados e boas práticas de desenvolvimento.

---

## 🚀 Funcionalidades

- **🔐 Autenticação:** Sistema de login seguro utilizando JWT (JSON Web Tokens).
- **🔑 Segurança:** Criptografia de senhas com a biblioteca `bcrypt`.
- **📦 CRUD de Posts:** Criação, leitura, atualização e exclusão de postagens.
- **👥 Gestão de Usuários:** Cadastro e listagem de usuários.
- **🔗 Relacionamentos:** Banco de dados relacional vinculando cada post ao seu respectivo autor.
- **✅ Validação:** Garantia de integridade dos dados enviados com `Joi`.

---

## 🛠️ Tecnologias Utilizadas

- **Runtime:** Node.js
- **Framework:** Express
- **Banco de Dados:** PostgreSQL
- **Deploy:** Render
- **Validação:** Joi
- **Segurança:** Bcrypt & JWT
